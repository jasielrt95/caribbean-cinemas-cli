# Security policy

## Supported versions

Security fixes are applied to the latest released version.

## Reporting a vulnerability

Report vulnerabilities through GitHub's private vulnerability-reporting
feature instead of a public issue. Include reproduction steps, affected
versions, and the expected impact when possible.

This project is a read-only client for public cinema data. It must not collect
Caribbean Cinemas credentials or payment information, create orders, or reserve
seats. Report any behavior that crosses that boundary as a security issue.
